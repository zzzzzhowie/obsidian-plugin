## Installation

### From Obsidian Community Plugins

1. Open **Settings** in Obsidian
2. Go to **Community Plugins** and disable Safe Mode
3. Click **Browse** and search for "Pinned Items"
4. Click **Install**, then **Enable**

### Manual Installation

1. Download `main.js`, `manifest.json`, and `styles.css` from the [latest release](https://github.com/YOUR-USERNAME/obsidian-pinned-items/releases)
2. Create a folder `YOUR-VAULT/.obsidian/plugins/pinned-items-plugin/`
3. Copy the downloaded files into this folder
4. Reload Obsidian
5. Go to **Settings â†’ Community Plugins** and enable "Pinned Items"
